## How to add a new post

1. Create a new file in the `_posts` folder named exactly like this:
   `YYYY-MM-DD-your-post-title.md`
2. Add this at the top of the file (called "front matter"):

```yaml
---
title: "Your Post Title"
date: 2026-07-20 09:00:00 +0530
excerpt: "One or two sentences that show up in the blog list."
---
```

3. Write your post below the front matter using normal Markdown — headings with `##`, **bold** with double asterisks, links with `[text](url)`, and so on.
4. Commit and push to GitHub. GitHub Pages rebuilds the site automatically — no other steps needed.

That's it. No build tools, no terminal commands required once your repo is set up.
